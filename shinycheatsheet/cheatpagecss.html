1. Color and Background

color

Sets the color of text.

background-color

Sets the background color of an element.

background-image

Sets an image as the background.

background-size

Defines the size of the background image.

opacity

Controls the transparency of an element.

2. Text and Font

font-family

Defines the font of text.

font-size

Sets the size of the font.

font-weight

Specifies the weight (boldness) of text.

line-height

Defines the space between lines of text.

text-align

Aligns text within its container.

text-decoration

Specifies decorations (underline, etc.) on text.

text-transform

Controls text capitalization.

letter-spacing

Adjusts the space between characters.

word-spacing

Adjusts the space between words.

3. Box Model

margin

Controls the space outside an element.

padding

Controls the space inside an element.

border

Sets the border around an element.

border-radius

Rounds the corners of an element.

box-shadow

Adds shadow around an element.

width

Sets the width of an element.

height

Sets the height of an element.

max-width

Defines the maximum width of an element.

max-height

Defines the maximum height of an element.

min-width

Defines the minimum width of an element.

min-height

Defines the minimum height of an element.

4. Positioning and Layout

display

Sets the display style of an element.

position

Positions an element (relative, absolute, fixed).

top

Defines the top position of an element.

right

Defines the right position of an element.

bottom

Defines the bottom position of an element.

left

Defines the left position of an element.

float

Positions an element to the left or right.

clear

Clears floated elements.

flex

Applies flexbox layout.

grid

Applies grid layout.

5. Visibility and Overflow

visibility

Controls visibility of an element.

overflow

Handles content overflow.

overflow-x

Handles horizontal content overflow.

overflow-y

Handles vertical content overflow.

6. Transitions and Animations

transition

Applies transition effects on property changes.

transition-duration

Sets the duration of a transition.

animation

Defines keyframe animations.

animation-duration

Sets the duration of an animation.

7. CSS Selectors

*

Universal selector, selects all elements.

div, p, h1, etc.

Type selector, selects specific HTML tags.

.classname

Class selector, selects elements with a specific class.

#idname

ID selector, selects an element with a specific ID.

:hover

Pseudo-class for hovering over an element.

:active

Applies styles to an element while it is being clicked.

:focus

Applies styles to an element when it is selected or focused.
