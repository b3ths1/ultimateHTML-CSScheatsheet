1. Basic Structure Tags

<html>

Root element of an HTML document.

<head>

Contains metadata, scripts, and links to external resources.

<title>

Sets the title of the webpage.

<body>

Contains the visible content of the page.

2. Content Organization Tags

<header>

Represents the introductory section of a page or section.

<footer>

Represents the closing section of a page or section.

<section>

Groups related content together.

<article>

Defines self-contained content that is independent and reusable.

<nav>

Contains navigation links.

<aside>

Represents content that is tangentially related to the main content.

<main>

Represents the main content of the page.

3. Text Formatting Tags

<h1> - <h6>

Defines headings, with <h1> being the highest and <h6> the lowest level.

<p>

Defines a paragraph.

<strong>

Displays text in bold with strong emphasis.

<em>

Displays text in italics with emphasis.

<b>

Makes text bold without added emphasis.

<i>

Makes text italic without added emphasis.

<br>

Inserts a line break.

<hr>

Inserts a horizontal rule (divider).

<span>

Wraps inline elements or text for styling.

4. List Tags

<ul>

Creates an unordered (bulleted) list.

<ol>

Creates an ordered (numbered) list.

<li>

Defines a list item.

5. Media Tags

<img>

Embeds an image on the page.

<audio>

Embeds audio content.

<video>

Embeds video content.

<iframe>

Embeds another webpage inside the current one.

<figure>

Groups media elements such as images or diagrams.

<figcaption>

Provides a caption for a figure.

6. Table Tags

<table>

Creates a table for data representation.

<tr>

Defines a row in a table.

<td>

Defines a cell in a table row.

<th>

Defines a header cell in a table.

<thead>

Groups header content in a table.

<tbody>

Groups the body content in a table.

<tfoot>

Groups the footer content in a table.

7. Form Tags

<form>

Creates a form to collect user input.

<input>

Creates a control for user input, such as a text field or checkbox.

<textarea>

Creates a multiline text input.

<button>

Creates a clickable button.

<select>

Creates a dropdown list of options.

<option>

Defines an option in a dropdown list.

<label>

Associates a label with a form control.

8. Miscellaneous Tags

<meta>

Provides metadata about the HTML document.

<link>

Links external resources such as stylesheets.

<style>

Defines internal CSS styles.

<script>

Embeds or links to JavaScript code.

<details>

Creates an expandable/collapsible content section.

<summary>

Defines a visible heading for the <details> element.

<progress>

Represents the progress of a task.

<meter>

Represents a scalar measurement within a known range.
